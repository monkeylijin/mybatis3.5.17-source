# 在职程序员 MyBatis 源码学习计划

## 🕐 现实时间安排

### 每日时间投入
- **工作日**: 1-1.5小时/天
- **周末**: 3-4小时/天
- **每周总计**: 11-15.5小时

### 学习时间分配建议

#### 工作日安排（周一至周五）
```
早上通勤时间: 30分钟 - 听技术播客、看技术文章
午休时间: 30分钟 - 阅读源码、做笔记
晚上下班后: 30-60分钟 - 实践编程、调试源码
```

#### 周末安排（周六、周日）
```
周六: 2-3小时 - 深入学习、实践项目
周日: 1-2小时 - 总结复习、写学习笔记
```

## 📅 调整后的学习计划（6个月版本）

### 第一阶段：基础准备（4周）

#### 第1-2周：Java基础强化
**每日投入**: 1小时
**学习内容**:
- 反射机制（30分钟/天）
- 动态代理（30分钟/天）

**实践安排**:
- 工作日：阅读和理解概念
- 周末：编写简单示例代码

#### 第3-4周：MyBatis使用熟练度
**每日投入**: 1小时
**学习内容**:
- 基础CRUD操作
- 动态SQL使用
- 缓存配置

**实践安排**:
- 工作日：配置和测试
- 周末：完成一个完整项目

### 第二阶段：源码入门（6周）

#### 第5-6周：整体架构理解
**每日投入**: 1.5小时
**学习内容**:
- 三层架构分析
- 核心组件关系
- 配置文件解析流程

**实践安排**:
- 工作日：阅读源码，画架构图
- 周末：跟踪一个查询的完整执行流程

#### 第7-8周：SqlSession和Configuration
**每日投入**: 1.5小时
**学习内容**:
- SqlSessionFactory创建过程
- SqlSession生命周期管理
- Configuration配置解析

**实践安排**:
- 工作日：分析源码，做笔记
- 周末：编写自定义配置解析器

#### 第9-10周：Mapper接口动态代理
**每日投入**: 1.5小时
**学习内容**:
- MapperProxy实现原理
- MapperMethod方法解析
- 参数处理机制

**实践安排**:
- 工作日：理解代理机制
- 周末：手写简单的Mapper代理

### 第三阶段：深度理解（8周）

#### 第11-12周：Executor执行器体系
**每日投入**: 1.5小时
**学习内容**:
- BaseExecutor基础执行器
- SimpleExecutor简单执行器
- CachingExecutor缓存执行器

**实践安排**:
- 工作日：分析执行器源码
- 周末：编写自定义执行器

#### 第13-14周：StatementHandler语句处理器
**每日投入**: 1.5小时
**学习内容**:
- RoutingStatementHandler
- PreparedStatementHandler
- 语句处理流程

**实践安排**:
- 工作日：理解语句处理机制
- 周末：实现自定义语句处理器

#### 第15-16周：ResultSetHandler结果集处理
**每日投入**: 1.5小时
**学习内容**:
- DefaultResultSetHandler
- 结果映射机制
- 延迟加载实现

**实践安排**:
- 工作日：分析结果处理源码
- 周末：编写结果集缓存处理器

#### 第17-18周：插件系统
**每日投入**: 1.5小时
**学习内容**:
- Interceptor拦截器接口
- Plugin代理实现
- InterceptorChain拦截器链

**实践安排**:
- 工作日：理解插件机制
- 周末：编写实用插件

### 第四阶段：实践应用（4周）

#### 第19-20周：综合实践
**每日投入**: 2小时
**学习内容**:
- 自定义分页插件
- 性能监控插件
- 多数据源支持

**实践安排**:
- 工作日：设计和实现
- 周末：测试和优化

#### 第21-22周：项目实战
**每日投入**: 2小时
**学习内容**:
- 简化版MyBatis实现
- 性能优化实践
- 问题排查技巧

**实践安排**:
- 工作日：编码实现
- 周末：测试和文档
