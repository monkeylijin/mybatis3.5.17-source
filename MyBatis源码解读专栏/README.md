# MyBatis 源码解读专栏

## 📚 专栏简介

本专栏旨在帮助有一定 Java 基础的开发者深入理解 MyBatis 的设计思想和实现原理。通过源码分析、实践案例和可视化图表，让读者从源码角度掌握 MyBatis 的核心机制。

## 🎯 学习目标

- 深入理解 MyBatis 的设计思想和实现原理
- 掌握 MyBatis 的核心组件和扩展机制
- 具备 MyBatis 性能优化和问题排查能力
- 能够开发 MyBatis 扩展组件
- 成为团队中的 MyBatis 技术专家

## 📖 专栏结构

### 第一部分：基础篇（4篇文章）
- 第1篇：MyBatis 整体架构设计
- 第2篇：配置系统深度解析
- 第3篇：SqlSession 的创建与生命周期
- 第4篇：Mapper 接口的动态代理机制

### 第二部分：核心篇（6篇文章）
- 第5篇：Executor 执行器体系详解
- 第6篇：StatementHandler 语句处理器
- 第7篇：ParameterHandler 参数处理机制
- 第8篇：ResultSetHandler 结果集处理
- 第9篇：MappedStatement 映射语句解析
- 第10篇：动态 SQL 实现原理

### 第三部分：高级篇（6篇文章）
- 第11篇：插件系统深度解析
- 第12篇：缓存机制源码分析
- 第13篇：事务管理源码解析
- 第14篇：类型转换系统
- 第15篇：反射模块优化
- 第16篇：日志系统集成

### 第四部分：实战篇（4篇文章）
- 第17篇：MyBatis 性能优化实践
- 第18篇：MyBatis 扩展开发
- 第19篇：MyBatis 与 Spring 集成原理
- 第20篇：MyBatis 最佳实践与踩坑指南
